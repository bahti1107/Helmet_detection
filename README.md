# Helmet Detection Project

This repository contains code for detecting helmets in images and videos using YOLOv8.  
Large datasets, videos, and output results are hosted on Google Drive due to GitHub size limits.

---

## Files
- `dataset.yaml` — dataset configuration  
- `src/` — Python scripts and notebooks  
- `metrics_evaluation.ipynb` — evaluation notebook  
- `app.py` — Streamlit interface  
- `yolo8.pt` — small YOLOv8 model file
